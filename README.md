# pokebag

  

Pokebag App for Ciputra Technical Test

  

## Getting Started

  
- Get all depedency
    `flutter pub get`
- run the apps
    `flutter run`

    

> Pokebag package stack
> -

 1. State management : vein
 2. Local Storage : get_storage
 3. Router : go_router
 4. Depedency Injection : get_it
